
Software name:
    Rhyscitlema Graph Plotter 3D

Software description:
    Draw any graph in a 3D virtual space.


To compile on Linux, first download the repositories:
    * ../algorithms
    * ../lib_std
    * ../libmfet
    * ../librodt
    * ../read_write_image_file

then execute: make

To run, execute:
    ./GraphPlotter3D example.rodt
